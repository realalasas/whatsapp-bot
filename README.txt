مشروع بوت واتساب بسيط (قروبات) — جاهز للتشغيل

ملفات المشروع:
- index.js       -> كود البوت
- package.json   -> إعدادات npm
- shipments.json -> قاعدة بيانات شحن بسيطة (JSON)
- auth_info_baileys/ -> سيُنشأ تلقائياً بعد أول تسجيل

تنصيب وتشغيل:
1) افتح الطرفية داخل مجلد المشروع.
2) نفّذ:
   npm install
   npm start
3) عند التشغيل سيولد الملف qr.png داخل المجلد، أو سيطبع رمز QR في الطرفية.
   افتح qr.png وامسحه باستخدام WhatsApp (الأجهزة المرتبطة > ربط جهاز).
4) بعد تسجيل الدخول ستبقى الجلسة محفوظة في 'auth_info_baileys'.

كيفية الاستخدام:
- داخل أي قروب أو محادثة: اذكر رمز الشحنة مثل:
  MSKU1234567
  أو
  AWB456789123
- سيقوم البوت بالرد بمعلومات الشحنة الموجودة في shipments.json.
